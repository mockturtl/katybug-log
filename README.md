# katybug-log #

XMLSocket connection to Powerflasher SOS using the Flex logging framework.

## dependencies ##

Requires [Flex framework](http://opensource.adobe.com/wiki/display/flexsdk/Downloads) v4.1 or greater.

## build ##

Use the [ExportSWC](http://sourceforge.net/projects/exportswc/) plugin for FlashDevelop.
